# HTML-DOCX Compiler Demos

This folder contains practical examples demonstrating how to use the html-docx-compiler package.

## Demos

### 1. Basic Usage (`basic-demo.js`)
Simple example showing how to convert HTML to DOCX with:
- Paragraphs and headings
- Text formatting (bold, italic, underline)
- Lists
- Links

### 2. Advanced Usage (`advanced-demo.js`)
More complex example demonstrating:
- Tables with styling
- Images from URLs
- Custom configuration
- Multiple formatting options
- Complex nested structures

## Running the Demos

```bash
# Run basic demo
node demo/basic-demo.js

# Run advanced demo
node demo/advanced-demo.js
```

Both demos will create DOCX files in the `demo/output/` folder.
